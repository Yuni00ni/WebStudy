- express : node.js 기반의 서버를 만들 수 있는 프레임워크
-- 프레임워크 : 서버 / 프론트엔드 등 빠르게 개발할 수 있는 틀

- npm : npm가 가지고 있는 도구 설치
- npx : 설치한 도구를 사용할 때

- app.js : express 코어

- <form> : 서버로 데이터를 전송할 수 있는 방법중 하나(입력양식)
=> action : 어디로 요청하는지 / name : 어떤 값을 보내는지 / submit : 제출버튼
+ method => form 태그에서는 get/post
get : 요청 데이터가 패킷의 head에 실어지기 때문에 노출됨
post : 요청 데이터가 패킷의 body에 실어지기 때문에 노출되지 않기에 보안상 상대적으로 우수하나 완벽하진 않기에 일반적으로 사용자가 입력한 값을 암호화 시킴
