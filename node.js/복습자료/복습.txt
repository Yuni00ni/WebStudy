- 웹서버 : 클라이언트, 서버, Request, Response
*HTTP 통신 : 클라이언트의 요청이 우선적으로 수행 -> 서버의 응답은 필수적으로 이루어져야함!

- Node.js : JavaScript를 단독적으로 실행할 수 있는 환경(-> 서버영역)
+ 싱글스레드를 기반으로 비동기 I/O 처리
  => 비동기 I/O(파일, DBL...) 오래 걸리는 작원은 libuv 라이브러리를 사용해서 별도의 쓰레도 풀에서 작업을 처리함
    => 동시에 작업이 처리되는 것 처럼 동작함

많이 사용되는 영역 : 스트리밍, 실시간 작업(채팅, 알림, 결제 ...)

- 모듈
1) http : 서버 정의
2) fs : 파일(읽기, 쓰기)
3) url : url 파싱(pathname, query)
